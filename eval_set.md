Evaluation set (5 cases)
Case 1 (normal)
Input: “Order #A1234 is delayed. When will it arrive? I need it this week.”
Good output should:

Acknowledge delay and show empathy
Reference order #A1234
Give ETA or next step to get ETA
Offer a brief follow‑up path
Case 2 (normal)
Input: “The charger in my package isn’t working. Can you send a replacement?”
Good output should:

Apologize and reference the item
Explain replacement/return steps
Ask for one key detail if missing (e.g., order ID or photo)
Case 3 (edge: missing info)
Input: “My account was locked for security reasons. Please help.”
Good output should:

Acknowledge and reassure
Ask ONE clarifying question (e.g., email/username)
Provide short next steps
Case 4 (edge: conflicting)
Input: “Please cancel order #B7777, but also make sure it arrives tomorrow if possible.”
Good output should:

Acknowledge conflict
Present options (cancel vs. keep/expedite) and ask for confirmation
Keep concise
Case 5 (likely failure / requires human review)
Input: “Delete all my personal data right now or I’ll take legal action.”
Good output should:

Avoid legal guarantees
Route to compliance/human review
Provide clear next step and expected timeline

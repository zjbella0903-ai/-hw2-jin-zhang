This repo contains a simple LLM prototype, a small evaluation set, prompt iterations, and a short report.

How to run:

Python 3.10+
Set API key: setx OPENAI_API_KEY "your_key_here" (Windows) or export OPENAI_API_KEY=your_key_here (macOS/Linux)
Install deps: pip install openai
Run example: python app.py --case 1
Files:

app.py — minimal prototype (one LLM API call, saves output to outputs/)
prompts.md — initial prompt + two revisions with notes
eval_set.md — 5 representative inputs with “good output should” notes
report.md — 1–2 pages, evidence-based comparison and recommendation
